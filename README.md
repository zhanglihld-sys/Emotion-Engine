# Emotion Engine

Quantitative market sentiment and stress monitoring system.

Generates a Bloomberg-style dashboard based on volatility, credit, liquidity and market breadth indicators.

Example dashboard:

![Dashboard](examples/sample_dashboard.png)

---

## Install

pip install -r requirements.txt

---

## Run

python emotion_engine.py

---

## Data Sources

Yahoo Finance  
FRED (Federal Reserve Economic Data)
